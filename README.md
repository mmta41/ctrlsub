# CtrlSub 

### subdomain take over tools


The project is inspired by [SubOver](https://github.com/Ice3man543/SubOver)



## Install
```
go get github.com/mmta41/ctrlsub
```


## Usage
```
./ctrlsub -l sub.host.com
```
